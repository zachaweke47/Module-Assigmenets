Module 1

Create a page that shows the main image of a recipe and five small images of other recipes. When clicking on each image, each image should take the user to the recipe of that image.

Module 2

Create a table without using styling. HTML code need to pass W3C validator. Iamge for the table should be hosted locally.

Module 3

Use an external CSS sheet to style content on a .HTML file. The page must incorporate external fonts, change heading styles, and colors. Both HTML and CSS code must pass validation.

Module 4

Use an external CSS to style content on a .HTML file. The page must incorporate external fonts, change heading styles, and colors. Also, use <div> to create different blocks & change specific parts of an HTML section. Finally, use lists to create a horizontal menu.

Module 5

Create a JavaScript program by asking a user for some values and returning results to the HTML page. Apply the concepts of variables, loops and prompting the user for a value.

Module 7

Modify the existing code of an example, and add a new object to the page using JavaScript.

Module 8

Using DOM and events, enable a user to add an element to an existing list.

Module 9

Use a baseline code to learn about even propagation and how to change the behavior of a default event.

Module 10

Review an existing site for usability and accessibility. Test the website against the W3C Validator, CSS Validator, and PageSpeed Insights. Note that this is a link to a word document.
